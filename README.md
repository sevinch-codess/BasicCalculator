
# Simple Shop Earnings Calculator 🛒

A Java console application that calculates a shop's total income and net profit based on product sales and expenses. This project was developed as part of the **JetBrains Academy** Java course.

## ✨ Features
- Lists prices for various products (Bubblegum, Ice Cream, etc.).
- Calculates total earned amount (Income).
- Takes user input for staff and other expenses.
- Calculates and displays the **Net Income**.

## 🛠 Tech Stack
- **Language:** Java 11+
- **Tools:** Scanner API, OOP Concepts (Classes & Methods).

## 🚀 How to Run
1. Ensure you have Java installed.
2. Clone this repository.
3. Run the following commands:
   ```bash
   javac src/calculator/*.java
   java calculator.Main
   ```

## 👨‍💻 Learning Goals
- Practicing basic Java syntax.
- Understanding how to handle user input with `Scanner`.
- Organizing code using Classes and Methods.
